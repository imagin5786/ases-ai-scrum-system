## 📁 Typical project flow

A normal flow looks like this:
1. Create or open a project.
2. Set the project goal.
3. Break the goal into tasks.
4. Run each task in order.
5. Review results.
6. Update the plan for the next step.

This helps reduce drift and keeps the project on track.
